# 🔐 Secure Password Generator

Cryptographically secure password generator that runs entirely in your browser.

## ✨ Features

- **Cryptographically secure** — uses `crypto.getRandomValues()` instead of `Math.random()`
- **Generate 8 passwords** at once for easy selection
- **Complexity toggle** — option to include/exclude special characters
- **One-click copy** — click on a password or the copy icon
- **Privacy-first** — all generation happens locally in your browser, no server requests
- **Responsive design** — works on desktop and mobile devices

## 🚀 Usage

1. Open `index.html` in any browser
2. Toggle "Special chars" if needed
3. Click on a password or the 📋 icon to copy
4. Click "Generate new" to create a fresh batch of passwords

## 🛠️ Tech Stack

- HTML5
- CSS3 (light green trust theme)
- Vanilla JavaScript (zero dependencies)
- Web Crypto API for secure generation
